Virtual Server

Virutal Server is a tool aim to help you for test your on hand IoT hardware. Both MQTT, HTTP API, UDP, and RR are supported.

Server Address:

MQTT : iot.dorony.com:1883

HTTP : iot.dorony.com:2424/api/v1/

UDP : iot.dorony.com:2323

$> mosquitto_pub -h 10.211.55.4 -t "rx" -u "coody" -P "liu383" -m "{\"token\":\"T7V511GS J5KP1KJY\", \"message\":\"Hi from mosquitto\"}"

$> curl -i -X POST -H "Content-Type:application/x-www-form-urlencoded" 10.211.55.4:2424/ api/v1/rx -d 'token=T7V511GSJ5KP1KJY&message=Hi from UDP client'

$> echo "{\"token\":\"T7V511GSJ5KP1KJY\", \"message\":\"Hi from UDP client\"}" | socat - udp4-datagram:10.211.55.4:2323
